# Tafahom Extension

Browser extension for translating YouTube videos into sign language using Tafahom's AI-powered avatar system.

## Features

- Extracts subtitles from YouTube videos
- Translates text into sign language animations
- AI-powered 3D avatar rendering
- Side panel integration for seamless experience

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output will be in the `dist/` directory ready for Chrome extension loading.
